# Customer Churn Prediction — End-to-End MLOps Project

An industry-grade machine learning project that predicts customer churn, built to
demonstrate a complete MLOps workflow: reproducible pipelines, experiment tracking,
model serving, CI/CD, orchestration, and monitoring.

## 🎯 Goals
- Build a reproducible, modular ML pipeline (not notebooks-in-production)
- Track experiments and version models with MLflow
- Serve the model as a REST API and containerize it with Docker
- Automate testing/builds with CI/CD, schedule retraining, and monitor for data drift

## 🛠️ Tech Stack
- **Language:** Python 3.12
- **Data & Modeling:** pandas, NumPy, scikit-learn, XGBoost (GPU-capable)
- **Experiment Tracking & Registry:** MLflow
- **Serving:** FastAPI + Uvicorn
- **Containerization:** Docker
- **CI/CD:** GitHub Actions
- **Orchestration:** Prefect
- **Monitoring:** Evidently
